# Contributing

This is a personal project. I maintain it for my own use and share it because others might find it useful.

## What's welcome

**Discussions** — Questions, ideas, and conversations are welcome in [GitHub Discussions](https://github.com/laat/mor/discussions). If something comes up that makes sense to act on, I will.

**Bug fixes** — If you've found a bug and have a fix, a PR is fine.

**Security fixes** — If you've found a security issue, please open a PR or contact me directly at sigurd@fosseng.net.

## What's not

**Feature PRs** — I'm unlikely to merge unsolicited feature work. The project reflects my own workflow and opinions. If you have an idea, start a discussion first.

**Issues** — Issue tracking is disabled. If something is broken, submit a PR.

## Fork freely

The code is MIT licensed. Fork it, change it, make it yours. That's the point of open source.
